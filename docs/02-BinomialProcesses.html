Coin Tossing

  1. Write a function which simulates tossing a “biased coin” (with \(p=0.157407\)) \(108\) times and computes the proportion of “heads”.

  2. Call this function “many” times and plot a histogram

def toss_biased_coin(pHead, n, seed=None):
  """
    Simulates n tosses of a biased coin and returns the proportion of heads,
    using rng.choice() for better conceptual clarity.

    Parameters
    ----------
    pHead : float
        Probability of heads (between 0 and 1).
    n : int
        Number of tosses.
    seed : int, optional
        Random seed for reproducibility.

    Returns
    -------
    float
        Proportion of heads observed.
    """
  rng = np.random.default_rng(seed)

  tosses = rng.choice([0,1], size=n, p=[1-pHead,pHead])
  cts = np.sum(tosses == 1)
  cts = np.sum(tosses)
  prop_heads = cts/n

  prop_heads = np.mean(tosses)

  return prop_heads
