11  Dimensions as “axis”

Row-wise vs. Column-wise operations

Averages

Here’s a quick example showing how axis works for row-wise vs column-wise averages in pandas.

import pandas as pd
import numpy as np

df = pd.DataFrame({
    'A': [1, 4, 7],
    'B': [2, 5, 8],
    'C': [3, 6, 9]
})

# Column-wise mean (down the rows)
col_means = df.mean(axis=0)

# Row-wise mean (across the columns)
row_means = df.mean(axis=1)

print("Column-wise means:")
print(col_means)

print("\nRow-wise means:")
print(row_means)

Output:

Column-wise means:
A    4.0
B    5.0
C    6.0
dtype: float64

Row-wise means:
0    2.0
1    5.0
2    8.0
dtype: float64
  • axis=0 → operate down columns → gives you column-wise results.
  • axis=1 → operate across columns → gives you row-wise results.

Think about data for which it is genuinely meaningful to compute both row-wise and column-wise averages. Here are the some candidates you can actually load directly in Python or R, with explanations why both directions matter.

12 1. US Arrests dataset (R datasets package)

Loadable via: from pydataset import data or datasets::USArrests

Structure: states × crime types

  • Rows: states
  • Columns: crime categories (Murder, Assault, UrbanPop, Rape)

Why row & column means make sense:

  • Row-wise averages: average crime intensity per state
  • Column-wise averages: average rate of each crime type across states

13 2. Retail or sales data (Python’s pandas-datareader or Kaggle CSVs)

Structure: region × year or product × month

  • Rows: regions/products
  • Columns: sales per year/month

Why row & column means:

  • Row-wise: average sales for each region/product
  • Column-wise: average sales per year/month across all regions

Example (hypothetical, but common): Superstore Sales dataset (Tableau sample), loadable from many public sources.

Horizontal vs. vertical concatenations

Below is a minimal example showing horizontal vs. vertical concatenation in pandas.

14 Example DataFrames

import pandas as pd

df1 = pd.DataFrame({
    'A': [1, 2],
    'B': [3, 4]
})

df2 = pd.DataFrame({
    'A': [5, 6],
    'B': [7, 8]
})

df3 = pd.DataFrame({
    'C': [9, 10]
})

15 Vertical concatenation (stack rows)

This is like placing one DataFrame under another.

vertical = pd.concat([df1, df2], axis=0)
print(vertical)

Result:

   A  B
0  1  3
1  2  4
0  5  7
1  6  8

16 Horizontal concatenation (stack columns)

This is like placing one DataFrame next to another.

horizontal = pd.concat([df1, df3], axis=1)
print(horizontal)

Result:

   A  B   C
0  1  3   9
1  2  4  10

17 Summary

  • Vertical concat (axis=0) → adds rows
  • Horizontal concat (axis=1) → adds columns
