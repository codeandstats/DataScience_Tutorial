PolynomialFeatures from scikit-learn can only be applied to numeric input. So if you have mixed data types (VERY common!), such as the diamonds dataset (which contains categorical columns like cut, color, clarity), you must first convert the categorical variables into numeric form (e.g., one-hot encoding), then apply PolynomialFeatures.

Below are the typical ways to do it.

Option 1: Apply PolynomialFeatures only to numeric columns

If you want interactions/polynomials only among numerical features (carat, depth, table, x, y, z), you can do:

Let us add one-hot encoding later and make a full pipeline out of it:

# --- Numeric pipeline ---
num_pipe = Pipeline([
    ("impute", SimpleImputer(strategy="median")),
    ("scale", StandardScaler()),
    ("poly", PolynomialFeatures(degree=2, include_bias=False))
])

# --- Categorical pipeline ---
cat_pipe = Pipeline([
    ("impute", SimpleImputer(strategy="most_frequent")),
    ("ohe", OneHotEncoder(drop="first"))
])

# --- Combine both ---
preprocess = ColumnTransformer([
    ("num", num_pipe, numeric_cols),
    ("cat", cat_pipe, cat_cols)
])

# --- Final model pipeline ---
model = Pipeline([
    ("prep", preprocess),
    ("linreg", LinearRegression())
])

# Fit model (price is the target)
X = df.drop(columns="price")
y = df["price"]

model.fit(X, y)

print("Model fitted successfully!")

8 Option 2: Apply PolynomialFeatures after encoding categorical features

If you want interactions between numeric and categorical (e.g., carat^2 × color_G), you need to:

  1. One-hot encode categoricals
  2. Apply polynomial features to the whole matrix

The safe and correct way is with a pipeline:

preprocess = ColumnTransformer([
    ("cat", OneHotEncoder(drop="first"), categorical_cols),
    ("num", "passthrough", numeric_cols)
])

pipe = Pipeline([
    ("prep", preprocess),
    ("poly", PolynomialFeatures(degree=2, include_bias=False))
])

X_poly = pipe.fit_transform(df)

print("new number of columns:", X_poly.shape[1])

import pandas as pd
import statsmodels.api as sm
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.pipeline import Pipeline

# Load diamonds
#df = sm.datasets.get_rdataset("diamonds", "ggplot2").data

X = df[["carat", "depth"]]   # same variables as in statsmodels formula
y = df["price"]

# Pipeline: interaction terms only
pipe = Pipeline([
    ("interaction", PolynomialFeatures(degree=2, include_bias=False, interaction_only=True)),
    ("model", LinearRegression())
])

pipe.fit(X, y)
