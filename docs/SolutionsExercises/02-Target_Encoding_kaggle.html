17  Target Encoding

  1. Load the small kagggle data
  2. Impute missing values for the categorical column.
  3. Apply target encoding with smoothing and CV-safety.
  4. Fit a linear regression on numerical features + target-encoded feature.
  5. Report R².

import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import r2_score
from sklearn.pipeline import Pipeline
from sklearn.impute import SimpleImputer
import category_encoders as ce

np.random.seed(42)

# -----------------------------
# 1. Create small synthetic data
# -----------------------------
data = {
    "num1": np.random.normal(size=20),
    "num2": np.random.normal(size=20),
    "cat": ['A', 'B', 'C', 'A', np.nan, 'B', 'C', 'D', 'D', np.nan,
            'A', 'C', 'B', 'D', 'E', np.nan, 'E', 'B', 'C', 'A'],
    "y": np.random.normal(size=20)
}

df = pd.DataFrame(data)

# Split into train/test
X = df[["num1", "num2", "cat"]]
y = df["y"]
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=0)

# -----------------------------
# 2. Build pipeline
# -----------------------------
# Impute missing categorical values with constant "__MISSING__"
cat_imputer = SimpleImputer(strategy="constant", fill_value="__MISSING__")

# Target encoding with smoothing
# smoothing parameter alpha=5 (adjustable)
target_encoder = ce.TargetEncoder(cols=["cat"], smoothing=5)

# Regression
reg = LinearRegression()

# Full pipeline
pipeline = Pipeline([
    ("imputer", cat_imputer),
    ("te", target_encoder),
    ("reg", reg)
])

# -----------------------------
# 3. Fit pipeline on training data
# -----------------------------
pipeline.fit(X_train, y_train)

# -----------------------------
# 4. Predict and evaluate
# -----------------------------
y_pred = pipeline.predict(X_test)
r2 = r2_score(y_test, y_pred)

print("R² on test set:", r2)

Notes

  1. Imputation: We fill missing categories with "__MISSING__".
  2. Target encoding: category_encoders.TargetEncoder automatically computes smoothed means for each category.
  3. Pipeline ensures no leakage — target encoding is fit only on the training data and applied to test data safely.
  4. Numerical features (num1, num2) are used directly.
  5. The R² output gives model performance on the test set.
